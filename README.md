# Agentic AI Projects on Google Cloud

This repository contains two types of projects:
- **guided_projects** — Built as part of the [Gen AI Academy APAC Edition](https://vision.hack2skill.com/event/apac-genaiacademy?utm_source=hack2skill&utm_medium=homepage), a Google Cloud initiative powered by Hack2skill
- **self_projects** — Independent explorations and experiments beyond the curriculum

For local development setup and workflow, see [process.md](process.md).
